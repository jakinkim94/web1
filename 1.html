HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HIML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appreance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to creat structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HIML elements are delineated by tages, written using angel brackets.
